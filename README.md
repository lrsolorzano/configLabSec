# Laboratorio de Seguridad Cibernética en GCP

## Descripción
Laboratorio completo para pruebas de seguridad y pentesting en Google Cloud Platform con 3 nodos:
- Kali Linux (Sistema atacante)
- Ubuntu Server 22.04 (Servidor vulnerable)
- Windows Server 2022 (Objetivo AD)

## Características
- Infraestructura automatizada con Terraform
- VPC lab-ciber completamente aislada
- Scripts de inicialización automática
- Reglas de firewall restrictivas
- Documentación completa

## Estructura
- /terraform - Código IaC
- /scripts - Scripts de automatización
- /docs - Documentación

## Requisitos
- Google Cloud SDK
- Terraform >= 1.0
- Créditos o billing en GCP
